
# Monster Battle Game in Python 

This is my first ever turn base game made in Python, using [pygame.ce](https://pyga.me). I followed the youtube tutorial of [@CearCode](https://www.youtube.com/@ClearCode).  
Link to the video :arrow_right: *<https://www.youtube.com/watch?v=8OMghdHP-zs>*

The goal was to have fun while learning how to make 2D games.

There is no Score manager yet, I apologize 😓 for that, but it will be added soon, because I really like this one 😊

Enjoy ! :grin:



## Installation 

1. Clone the repository:
```bash
 git clone https://github.com/SergeMaikere/monster-battle.git
```

2. Install dependencies:
```bash
 pip install -r requirements.txt
 ```

## Usage 

To run the project use the following command:
```bash
python main.py
 ```
